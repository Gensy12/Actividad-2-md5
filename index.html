Tema 2. Define los elementos que componen la estructuración de una página web:

1. Contenedor (wrapper)

El contenedor actúa como un elemento estructural que agrupa otros componentes de la página, facilitando la organización y la aplicación de estilos. Permite centrar el contenido y controlar su ancho máximo, contribuyendo a un diseño limpio y ordenado.

2. Cabecera (header)

La cabecera es la sección superior del sitio donde generalmente se ubica el logotipo, el título del sitio y la navegación principal. Es fundamental para la identificación del sitio y ofrece accesibilidad a las secciones más importantes.

3. Contenido (content)

El contenido es el núcleo de la página web, donde se presenta la información principal y relevante para los usuarios. Esto incluye texto, imágenes, videos y otros elementos interactivos, diseñados para captar la atención y facilitar la comunicación del mensaje.

4. Menú (menu)

El menú es un componente esencial de navegación que permite a los usuarios moverse entre las distintas secciones del sitio. Su diseño debe ser claro y accesible, asegurando que los visitantes puedan encontrar fácilmente la información que buscan.

5. Pie (footer)

El pie de página se encuentra en la parte inferior del sitio y proporciona información adicional como derechos de autor, enlaces a políticas legales y redes sociales. Es útil para incluir detalles que complementan la experiencia del usuario sin interferir con el contenido principal.

6. Lateral (sidebar)

La barra lateral se utiliza para mostrar contenido adicional relacionado, como enlaces, anuncios o widgets. Su ubicación a un lado del contenido principal permite ofrecer información complementaria sin distraer de la narrativa central de la página.

7. Etiqueta div

La etiqueta div es un contenedor genérico que se utiliza para agrupar elementos en un diseño web. Aunque carece de significado semántico, es extremadamente útil para estructurar el contenido y aplicar estilos CSS de manera efectiva.

Tema 3. Identifica y describe los elementos que componen la estructuración semántica de una página web.

La estructuración semántica de una página web incluye el uso de etiquetas HTML semánticas como header, nav, main, article, section, aside y footer, que organizan el contenido. Se utiliza una jerarquía de encabezados para facilitar la lectura, y se incorporan atributos de accesibilidad y enlaces internos y externos para mejorar la navegación. Además, las metaetiquetas en el head y el uso de microdatos enriquecen el SEO, creando páginas comprensibles y accesibles.
